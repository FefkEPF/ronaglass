# Website Restoration & Modernization Skill

## Purpose
This skill provides a systematic approach to extracting content, media assets, branding, and structure from a legacy website and transforming it into a state-of-the-art cinematic web experience.

## Workflow & Strategy

1. **Content & Asset Extraction**:
   - Use `browser_subagent` and `read_url_content` to inspect the target website.
   - Extract page structure, headings, body text, service lists, contact details, social links, and media assets.
   - Map legacy navigation and content hierarchy to modern full-screen sections.

2. **Design System & Aesthetics**:
   - Integrate extracted content into a sleek, luxury design system.
   - Utilize dark themes, gold accents (`#c8a45c`), glassmorphism backdrop filters, elegant Google Typography (e.g. *Outfit*, *Playfair Display*).
   - Ensure responsive layout across desktop, tablet, and mobile devices.

3. **Performance & Smooth Scrolling**:
   - Maintain Lenis smooth scrolling engine for buttery 60+ FPS motion.
   - Seamlessly blend the 130-frame cinematic canvas sequence with content overlays and interactive sections.

4. **SEO & Accessibility**:
   - Preserve semantic HTML (`<header>`, `<main>`, `<section>`, `<footer>`).
   - Retain title tags, meta descriptions, alt attributes for images, and active phone / WhatsApp / contact links.
